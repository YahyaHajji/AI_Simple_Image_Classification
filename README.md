# AI Simple Image Classification

## Description
This project implements a simple image classification model using machine learning techniques. The goal is to classify images into predefined categories, showcasing the fundamentals of image classification with a straightforward approach.

## Installation

To set up the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/YahyaHajji/AI_Simple_Image_Classification.git
   cd AI_Simple_Image_Classification
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To train the image classification model and evaluate its performance, run the following command:

```bash
python main.py
```

You can adjust hyperparameters and model architecture in the `main.py` file as needed.

## Dataset

This project uses a dataset of images categorized into different classes. Ensure that the dataset is organized correctly as specified in the code.
